== Description ==
			  
Alle News stellt eine Auswahl an aktuellen, deutschsprachigen Nachrichten aus Deutschland, Österreich und der Schweiz 
zusammen und zeigt sie in der Sidebar an.

Alle News displays a selection of German breaking news from Germany, Austria and Switzerland in the sidebar.

== Installation ==

1. Laden Sie die Datei allenews.php in das Plugin-Verzeichnis (normalerweise 'wp-content/plugins/')
2. Melden Sie sich im Backend an
3. Rufen Sie den Punkt 'Plugins' auf
4. Aktivieren Sie das 'Alle News' Plugin
5. Rufen Sie den Punkt 'Widgets' auf
6. Ziehen Sie das Widget 'Alle News' in die Sidebar

Fertig! Ihr Blog zeigt nun aktuelle Nachrichten an.

Wenn Sie möchten, konfigurieren Sie das 'Alle News' Widget nach Bedarf. 


1. Upload allenews.php to your plugins folder (usually 'wp-content/plugins/')
2. Log-in to the backend
3. Go to the 'Plugins' page
4. Activate the 'Alle News' plugin
5. Go to the 'Widgets' page
6. Drag and drop the 'Alle News' widget to your sidebar

That's all! Your blog is displaying breaking news.

If neccessary, you can change the options of the widget.
